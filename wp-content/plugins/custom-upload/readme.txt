== Description ==
A simple plugin to upload any type of image in wordpress. We can use it to manage header gallery section.

== Installation ==
= How to install Custom Upload Plugin (manual) =
1. Download Custom Upload Plugin.
2. Upload it into your plugins directory from Plugin Add Feature or FTP.
3. It will create a directory /wp-content/plugins/custom-upload/
4. Active Custom Upload Plugin.

== Frequently Asked Questions ==
Que. How can we use it in Header Gallery  
Ans. After installing and activating this plugin, it will create Custom Upload Menu in admin.  
 You can create category for you images, and add images for that category.  
 You can write custom select query to fetch records to arrange it in custom wordpress template page.
